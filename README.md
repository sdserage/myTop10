# My Top 10

## About
My Top 10 is a simple application that allows users to create their own ranked lists.

## Developement Setup

### 1. Start MongoDB

`mongod --dbpath <path to data directory>`

`mongo`

### 2. Start Node Server

`cd server`

`nodemon`

### 3. Start NextJS

`cd client`

`npm run dev`

## Testing

### Test Server

`cd server`

`npm test`
